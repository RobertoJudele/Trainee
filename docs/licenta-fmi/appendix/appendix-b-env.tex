\section{Recomandari de operare}

\begin{itemize}[leftmargin=1.2cm]
    \item Fisierele \texttt{.env} reale nu se versioneaza in repository.
    \item In client se folosesc doar chei publice, iar cheile private raman exclusiv in backend.
    \item Pentru credentialele critice se defineste o procedura de rotatie periodica.
    \item In productie este recomandata folosirea unui secret manager dedicat.
\end{itemize}
